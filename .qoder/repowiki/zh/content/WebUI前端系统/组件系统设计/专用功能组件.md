## 简介
本文件面向 VAPT3 的专用功能组件，围绕消息渲染、富文本与代码高亮、媒体预览、通知与状态展示、认证与受保护路由、实时事件流以及设置管理等关键 UI 组件进行系统化技术说明。内容涵盖渲染逻辑、样式定制、交互流程、性能优化策略与可维护性设计，并提供可视化图示帮助理解组件间协作。

## 项目结构
这些组件主要位于 webui/src/components 及其子目录下，采用按功能域分层组织：消息类（MessageBubble、MessageList）、编辑与输入（Composer）、富文本与代码（MarkdownText、MarkdownTextRenderer、CodeBlock）、媒体与灯箱（ImageLightbox）、通知与状态（NotificationPanel、TaskStatusBadge、ConnectionBadge）、确认与空态（DeleteConfirm、EmptyState）、安全与路由（ProtectedRoute、AuthForm）、实时事件（ActivityEventStream）、设置视图（SettingsView）以及国际化语言切换（LanguageSwitcher）。组件之间通过 props、自定义事件与共享状态钩子（如 useNotifications、useActivityStream、ClientProvider）协同工作。

## 核心组件
- 消息渲染与交互：MessageBubble 负责单条消息的渲染、复制、工具调用轨迹展开、媒体与图片附件展示；MessageList 提供自动贴底滚动、近底部检测与“回到底部”按钮。
- 富文本与代码高亮：MarkdownText 作为入口，MarkdownTextRenderer 使用 react-markdown、remark-gfm、remark-math、rehype-katex 渲染，CodeBlock 集成 react-syntax-highlighter 并支持复制与深色/浅色主题切换。
- 媒体与预览：ImageLightbox 提供全屏图片浏览、键盘导航、相邻图片预解码、GPU 合成层与无障碍支持。
- 通知与状态：NotificationPanel 管理通知列表、未读计数、标记已读、错误与加载态；TaskStatusBadge/ConnectionBadge 分别用于任务状态与连接状态的语义化展示。
- 安全与路由：ProtectedRoute 与 AuthForm 实现基于引导状态的受保护路由与认证表单；ErrorBoundary 提供顶层错误捕获与恢复。
- 实时与设置：ActivityEventStream 展示实时事件流并支持刷新；SettingsView 管理模型、提供商、Base URL、API Key 与语言设置，包含保存、模型拉取与错误提示。

## 详细组件分析

### MessageBubble 消息气泡
- 渲染策略
  - 用户消息：右对齐胶囊样式，支持文本、图片与非图片媒体附件；图片使用 UserImages 预览，支持 Lightbox 查看。
  - 助手消息：左对齐卡片，Markdown 文本渲染；流式输出显示打点光标；非流式时显示复制按钮。
  - 追踪行（trace）：默认折叠，点击展开工具调用/进度轨迹。
- 媒体与附件
  - 视频附件直出 video 标签；文件附件直链下载或仅展示占位；图片缩略图懒加载与焦点态。
- 无障碍与动画
  - 复制反馈（复制/已复制）；键鼠交互与标签；进入动画与无障碍描述。

### MessageList 消息列表
- 自动贴底滚动：在用户位于底部时，新消息即时滚动到底部；流式期间禁用平滑滚动避免动画冲突；稳定后启用平滑滚动。

- 视觉增强：顶部与底部渐隐遮罩，提升滚动体验。

## 性能考量
- 消息列表滚动
  - 仅在 near-bottom 时自动滚动，避免频繁重排；流式期间禁用平滑滚动减少动画抖动。
- 代码高亮
  - 仅在需要时渲染 CodeBlock；主题切换通过 DOM class 观察，避免不必要的重渲染。
- 媒体预览
  - ImageLightbox 预加载相邻图片；translate3d 启用硬件加速；滚动区域使用 GPU 合成层。
- Markdown 渲染
  - 将重型渲染器拆分为独立 chunk，但为避免首帧闪烁，MarkdownText 直接静态导入渲染器。
- 通知与事件流
  - 面板打开时主动刷新；事件流支持错误态与重试，避免长时间挂起。

## 故障排查指南
- 错误边界
  - 出现渲染错误时，ErrorBoundary 会显示错误信息与重试按钮；开发者可在 onError 中接入遥测。
- 通知面板
  - 网络错误时显示错误提示与重试按钮；检查 token 是否有效与网络连通性。
- 设置视图
  - 加载失败弹窗提示；可重试或登出重新认证；确保 Base URL 与 API Key 正确。
- 事件流
  - 错误态显示暂停与重试；检查客户端连接状态与订阅权限。
- 代码复制
  - 若复制无效，检查浏览器剪贴板权限与 HTTPS 环境。
